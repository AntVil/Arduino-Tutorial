Genereller Aufbau



Ein Arduino besitzt viele Pins welche bestimmt Eigenschaften haben.
  • Der pin GND steht für Ground und ist wie der Minuspol
  • Die pins 3.5V und 5V sind wie Pluspole
  • Die pins 0 bis 13 (im Bild oben) sind Digitale pins
    • Bei diesen pins kann die Stromspannung an und ausgeschalten werden
    • Diese pins können auch auslesen ob eine Stromspannung anliegt
  • Die pins 0 bis 5 (im Bild unten) sind Analoge pins
    • Bei diesen pins kann die Stromspannung zu einem beliebigen Wert gesetzt werden
    • Diese pins können auch auslesen wie groß eine Stromspannung ist die anliegt


void setup(){

}

void loop(){

}


Ein Arduino Programm besteht immer aus mindestens diesem Code.
Jedes Programm beginnt zwischen den geschwungenen Klammern nach setup.
Dieser Code wird einmal beim start des Programms ausgefürt.
Der Code zwischen den geschwungenen Klammern nach loop wird immer wieder ausgeführt.

Eine LED ansteuern

void setup(){
  pinMode(7, OUTPUT);
}

void loop(){
  delay(1000);
  digitalWrite(7, HIGH);
  delay(1000);
  digitalWrite(7, LOW);
}


In diesem Beispiel wird eine LED jede Sekunde an und aus gemacht.
Dazu wird in setup mit pinMode dem Arduino gesagt, dass der pin 7 ein OUTPUT ist.
Dadurch können wir ab jetzt den Strom an pin 7 mit digitalWrite an (HIGH) und ausschalten (LOW).
Mit delay können wir dem Arduino sagen, dass er eine bestimmte Anzahl an Milisekunden warten soll.

Baue die Schaltung nach und ändere die delay's im Code und schau was passiert.

Zwei LED's ansteuern

void setup(){
  pinMode(6, OUTPUT);
  pinMode(7, OUTPUT);
}

void loop(){
  delay(1000);
  digitalWrite(6, HIGH);
  digitalWrite(7, LOW);
  delay(1000);
  digitalWrite(6, LOW);
  digitalWrite(7, HIGH);
}


In diesem Beispiel werden zwei LED's abwechselnd jede Sekunde an und aus gemacht.
Dazu wurde das Beispiel von davor leicht abgeändert.
Hier wird einfach ein weiterer pin (6) zu einem OUTPUT gemacht.
Jetzt können wir die beiden LED's abwechselnd mit digitalWrite an und ausmachen.

Knopf Eingabe

void setup(){
  pinMode(7, OUTPUT);
  pinMode(8, INPUT);
}

void loop(){
  if(digitalRead(8) == HIGH){
    digitalWrite(7, HIGH);
    delay(1000);
    digitalWrite(7, LOW);
  }
}


In diesem Beispiel leuchtet eine LED eine Sekunde lang, wenn man den Knopf drückt.
Dazu wird in setup dem Arduino gesagt, dass pin 8 ein INPUT ist
Jetzt können wir prüfen mit digitalRead auslesen, was für eine Stromspannung an diesem pin ist.
Mit if können wir prüfen ob etwas bestimmtes zutrifft und jenachdem bestimmten Code ausführen.
In diesem Beispiel wir die Stromspannung an dem pin ausgelesen und mit == geprüft ob diese gleich HIGH ist.
Falls ja wird die LED an pin 7 für eine Sekunde an und dann wieder aus gemacht.
